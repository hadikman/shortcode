--- title: list all pages description: This command is based on the "wp post list" used to return all posts but the "--post_type=page" flag changes the search scope to return only pages. By default the output if formatted as a table and includes page ID, title, slug, creation date, and the status. tags: wordpress cli date: 2021-10-14 ref: https://developer.wordpress.org/cli/commands/post/list/ --- wp post list --post_type=page
